PICT B.E. Course Structure

Explore the subjects and curriculum for all 4 years of the B.E. program at PICT.

First Year (FE)

Engineering Mathematics I: Basics of calculus, differential equations, and linear algebra.

Engineering Physics: Fundamental concepts of optics, semiconductors, and mechanics.

Engineering Chemistry: Atomic structure, bonding, and engineering applications of materials.

Basic Electrical Engineering: Electrical circuits, Kirchhoff's laws, and transformers.

Programming in C: Introduction to problem-solving using C language.

Second Year (SE)

Data Structures & Algorithms: Arrays, stacks, queues, linked lists, trees, and graphs.

Computer Organization: Basic computer architecture, memory, and I/O systems.

Object-Oriented Programming: Concepts of OOP using C++/Java, including classes and inheritance.

Discrete Mathematics: Sets, logic, relations, graph theory, and combinatorics.

Digital Logic Design: Boolean algebra, flip-flops, counters, and combinational circuits.

Third Year (TE)

Operating Systems: Process management, memory, file systems, and synchronization.

Database Management Systems: SQL, relational models, normalization, and transactions.

Software Engineering: SDLC models, requirements, design patterns, and testing.

Computer Networks: OSI model, TCP/IP, routing, and wireless communication basics.

Web Technologies: HTML, CSS, JavaScript, and web application frameworks.

Final Year (BE)

Machine Learning: Supervised and unsupervised learning, neural networks, and applications.

Cloud Computing: Cloud architecture, services, virtualization, and deployment models.

Information Security: Encryption, network security, authentication, and ethical hacking.

Electives (AI, Blockchain, IoT): In-depth learning on emerging technologies and their applications.

Project Work: Capstone project integrating knowledge across the curriculum.
